AR Dinosaur Wikipedia

Overview

This is a Dinosaur Wikipedia Application demonstrating three dinosaurs with information in text and audio. This application aims to provide a new way of reading information and learning about dinosaurs.

Features

A home menu presents the start button and a background image of dinosaurs. It is the first start of the application that prepares users for what's coming next. The start button in the center of the screen will guide users to the next page where the learning happens. The quit icon will quit the application if pressed. The info icon will expand a panel showing the information about the dinosaurs. In the information panel, three buttons are the play icon, pause icon and stop icon, which will function as they imply.

Technology

Augmented Reality; Unity 3D; ARcore; Vuforia SDK
