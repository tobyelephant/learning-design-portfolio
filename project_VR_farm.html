VR Game: Excellent Farm

Overview

This project is to develop a farm simulation game that users can collect vegetables and feed the milk cow. The scoring system is simple. As long as users feed the milk cow, put vegetables in the container or give them to the cowboy, they will get one point for each vegetable.

Design Document

Features

This fully immersive Virtual Reality game is developed for Oculus Quest 2. Users can teleport to move in the game. They can use the controller to grab the vegetables and throw them to feed the cow. The vegetables are located randomly in space once the game starts, rotating and floating in the air. This way, more space is used as users need to look around to collect vegetables.

Technology

Virtual Reality; Unity 3D; C#; XR Origins; 3D Animation
